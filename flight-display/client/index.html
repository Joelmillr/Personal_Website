About This Display

What is this? This is a pilot Helmet-Mounted Display (HMD) system I developed for aerospace flight test operations. The HMD overlays real-time flight symbology onto the pilot's field of view, driven by live sensor data from the aircraft. This page replays an actual test flight using recorded data.

What you're seeing:

  • 3D View (Top): The HMD itself — the symbology and orientation cues rendered in real time that a pilot would see while wearing the display during flight
  • Video Feed (Left): Cockpit footage recorded during the test flight, synchronized to the HMD playback
  • Flight Path (Right Top): The aircraft's GPS track over the course of the test flight
  • Altitude Profile (Right Bottom): Aircraft altitude over time
  • Attitude Profile (Bottom): Aircraft orientation (pitch, roll, heading) throughout the flight

How to use: Use the controls at the bottom to jump to specific flight maneuvers, or let the playback run automatically. All panels are synchronized in real time.
